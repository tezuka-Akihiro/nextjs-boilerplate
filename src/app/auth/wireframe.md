## フォーム要素仕様

### 入力フィールド共通
- **高さ**: 48px（モバイル: 44px）
- **幅**: フォーム幅100%（最大400px）
- **マージン**: 下16px
- **ボーダー**: 1px solid `var(--border-color)`
- **フォーカス**: `var(--primary-600)` ボーダー

### ボタン仕様
```
プライマリボタン（ログイン・登録）:
├─ 背景: var(--primary-600)
├─ テキスト: white
├─ 高さ: 48px
└─ 幅: 100%

ソーシャルログインボタン:
├─ 背景: white / var(--surface-100)  
├─ ボーダー: 1px solid var(--border-color)
├─ 高さ: 48px
└─ アイコン + テキスト
```

## レスポンシブ対応

### ブレークポイント
- **Desktop**: 1024px以上 - 中央配置カード（最大400px幅）
- **Tablet**: 768px-1023px - 中央配置カード（最大400px幅）  
- **Mobile**: 767px以下 - 全幅フォーム（左右16px余白）

### カード配置
```
Desktop/Tablet:
- 中央配置（画面中央）
- カード幅: 400px
- 上下余白: 120px

Mobile:  
- 上部配置
- 左右余白: 16px
- 上余白: 40px
```

## 背景・装飾

### 背景デザイン
- **ベース色**: `var(--background-subtle)`
- **パターン**: なし（シンプルな単色）
- **ブランドカラー**: ヘッダーロゴのみ

### フォームカード
- **背景**: `var(--surface-primary)`
- **影**: `box-shadow: 0 8px 32px rgba(0,0,0,0.08)`
- **角丸**: `border-radius: 12px`
- **パディング**: 32px（モバイル: 24px）

## 画面遷移フロー

### 認証フロー
```
ログイン画面
├─ 成功 → メイン画面
├─ 失敗 → エラー表示（同画面）
└─ 新規登録リンク → 登録画面

新規登録画面  
├─ 成功 → プロフィール設定画面
├─ 失敗 → エラー表示（同画面）
└─ ログインリンク → ログイン画面
```

### パスワードリセット
```
ログイン画面 "パスワードを忘れた"
└─ パスワードリセット画面
   ├─ メール送信成功 → 確認画面
   └─ 戻るリンク → ログイン画面
```
